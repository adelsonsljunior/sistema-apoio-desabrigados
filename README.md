# sistema-apoio-desabrigados

## Como rodar

### 1. Configure as variáveis de ambiente

> Crie uma cópia do arquivo `.env.example` com o nome de `.env`.

~~~bash
$ docker compose up -d
~~~

### 2. Suba o  banco de dados

~~~bash
$ docker compose up
~~~

### 3. Execute a aplição usando uma IDE de sua escolha
